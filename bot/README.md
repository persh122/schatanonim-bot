# 🤖 Schatanonim Bot

Анонимный Telegram-бот для случайного общения с незнакомцами.

## Возможности

- 🔎 Случайный подбор собеседника (с очередью ожидания)
- 💬 Пересылка текста, фото, видео, голосовых, стикеров и файлов
- ⏭ Кнопки «Следующий» и «Завершить чат»
- 💎 VIP через Telegram Stars:
  - Выбор пола собеседника
  - Приоритет в очереди
  - Без рекламы
- 🛠 Админ-панель: статистика, бан/разбан, рассылка, выдача VIP
- 🗄 База данных SQLite

## Быстрый старт

### 1. Получить токен бота

1. Напишите [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте `/newbot` и следуйте инструкциям
3. Скопируйте токен вида `123456:ABC-DEF...`

### 2. Установить зависимости

```bash
cd bot
pip install -r requirements.txt
```

### 3. Настроить переменные окружения

Создайте файл `bot/.env` (или задайте переменные окружения):

```env
TELEGRAM_BOT_TOKEN=ваш_токен_от_BotFather

# ID администраторов (через запятую)
ADMIN_IDS=123456789,987654321

# Путь к базе данных (по умолчанию bot/data/bot.db)
# DB_PATH=bot/data/bot.db
```

Чтобы узнать свой Telegram ID, напишите [@userinfobot](https://t.me/userinfobot).

### 4. Запустить бота

```bash
cd bot
python main.py
```

## Настройка VIP

VIP продаётся за **Telegram Stars** (⭐). Параметры в `config.py`:

| Параметр          | По умолчанию | Описание                    |
|-------------------|--------------|-----------------------------|
| `VIP_PRICE_STARS` | `100`        | Цена в Stars                |
| `VIP_DURATION_DAYS` | `30`       | Срок VIP в днях             |
| `AD_INTERVAL`     | `20`         | Реклама каждые N сообщений  |

## Админ-панель

Добавьте свой Telegram ID в `ADMIN_IDS` и отправьте `/admin`.

Доступные функции:
- 📊 Статистика бота
- 🚫 Бан/разбан пользователей
- 💎 Выдать VIP вручную
- 📣 Рассылка всем пользователям
- 👥 Список последних пользователей
