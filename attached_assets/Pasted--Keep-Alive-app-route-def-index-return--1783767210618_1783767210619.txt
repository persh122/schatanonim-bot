def webhook():

        json_str = request.get_data().decode('UTF-8')
        update = telebot.types.Update.de_json(json_str)
        bot.process_new_updates([update])
        return "OK", 200
    except Exception as e:
        print(e)
        return "ERROR", 500

Инструкция по запуску на Replit

1. Создайте Repl: Зайдите на Replit, создайте новый Python Repl.
2. Вставьте код: Удалите всё из main.py и вставьте код сверху.
3. Установите библиотеку: В разделе "Shell" (консоль) внизу выполните команду:
     pip install pyTelegramBotAPI flask
   4. Получите токен: Напишите @BotFather в Telegram, создайте нового бота командой /newbot и скопируйте полученный токен.
5. Добавьте секрет: В Replit нажмите на иконку "Замок" (Secrets). Создайте новый секрет с ключом BOT_TOKEN и значением вашего токена.
6. Запустите: Нажмите кнопку "Run". Бот будет работать и отвечать на команды.
